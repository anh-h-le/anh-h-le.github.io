Google's Project Sunroof Data Querying and Visualization

Overview

The TAC 249 (Introduction to Data Analytics) project aimed at demonstrating familiarity with data analytics and visualization software and languages through analyzing the Google's Project Sunroof data.

Data description:

  • project-sunroof-state.csv: aggregated U.S. state-level solar potential data, including building counts, qualification rates, and total sunlight generation estimates from Project Sunroof.
  • region lookup table.csv: reference table mapping each U.S. state to its corresponding region.

Tools: MySQL Workbench (SQL), MongoDB (NoSQL), Tableau.

Part 1: Data Querying with MySQL

  • Imported the datasets into the database by creating two destination tables with primary keys and attributes.
  • Queried the data using SQL, including joins, unions, subqueries, and functions, to explore highest-value entries and summary statistics.

Part 2: Data Querying with MongoDB

  • Exported the resulting table from joining two tables in part 1 as a JSON file and imported the file into MongoDB.
  • Queried the data using MongoDB Query Language (MQL) to retrive aggregated data based on regions, states, or roof orientations.

Part 3: Data Visualization with Tableau

Seoul's Bike Rentals Data Visualization and Machine Learning

Overview

The TAC 216 (Applied Python) final project required creating a program to manipulate and visualize a Big Data dataset, showcasing skills in scientific computing, machine learning, and data visualization.

Data description: seoul_bike_sharing.csv

  • Source: UC Irvine Machine Learning Repository
  • Period: 12/01/2017-11/30/2018
  • Contains data about date, hour, weather, season, whether it was a holiday, and whether it was a functioning day.

Tools: Python (pandas, numpy, matplotlib, scikit-learn, sqlite3).

Data Integration and Cleaning

  • Loaded and stored data in SQLite, ensuring unique (date-hour) records and a reproducible ETL pipeline.
  • Cleaned and standardized data using Pandas.

Interactive Exploration and Visualization

  • Built a menu-driven interface that lets users select visualization options dynamically.
  • Used Matplotlib to generate visualizations:
    • Average hourly rentals.
    • Rentals by season.
    • Rentals on holidays vs. non-holidays
    • Daily rental trends over the year.

Predictive Modeling

  • Trained an MLP classifier to predict seasons from weather data and evaluated with confusion matrix and classification report.
